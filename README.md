# anbudev-web

Portfolio website for Anbuselvan T. — Backend-focused AI & Data Engineering student at JAIN (Deemed to be University).

Live at: [anbudev.me](https://anbudev.me)

## Tech Stack

- **Framework**: Next.js 16 (App Router, Static Export)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Font**: Inter (Google Fonts)

## Sections

1. **Hero** — Headline, sub-headline, GitHub/LinkedIn CTAs, key stats
2. **Technical Stack** — Categorized skill grid (Core Depth & Breadth)
3. **Featured Projects** — Argo FloatChat, Project Aegis, Project Healer
4. **Professional Experience** — Axiom Digital, Operational Manager
5. **Education & Contact** — JAIN B.Tech details, contact form, social links

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
